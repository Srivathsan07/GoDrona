ESC Calibration

Click Calibrate to start, then:

  • Disconnect USB and battery so flight controller powers down
  • Connect the Battery
  • The arming tone will be played (if the vehicle has buzzer attached)
  • If using flight controller with a safety button press it until it displays solid red
  • You will hear musical tone then two beeps
  • Few seconds later you should hear a number of beeps (One for each battery cell you are using)
  • And finally a single long beep indicating the end points have been set and the ESC is calibrated
  • Disconnect the battery and power up again normally
